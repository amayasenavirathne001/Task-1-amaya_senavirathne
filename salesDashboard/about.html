Project Overview

This project presents a responsive sales analytics dashboard developed using HTML, CSS and JavaScript. The dashboard provides insights into sales performance through interactive tables and summary statistics.

Dataset Description

The dataset contains retail sales transactions including:

  • Date
  • Region
  • Product
  • Quantity
  • Unit Price
  • Total Price
  • Salesperson
  • Payment Method
  • Promotion
  • Customer Type
  • Region Manager

Features

  • Total Revenue Summary
  • Total Orders Calculation
  • Total Products Sold
  • Top Performing Region
  • Product Revenue Analysis
  • Regional Sales Analysis
  • Responsive Layout Design

Technologies Used

  • HTML5
  • CSS3
  • JavaScript (ES6)
  • Responsive Web Design
